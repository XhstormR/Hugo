构建 Lua

2020-03-15

Updated on 2020-03-15

https://www.lua.org/download.html

Step

pacman -Syu
pacman -S --noconfirm --needed gcc make

curl -O https://www.lua.org/ftp/lua-5.4.0.tar.gz
tar zxf lua-5.4.0.tar.gz
cd lua-5.4.0
make posix install CC='x86_64-pc-msys-gcc'
