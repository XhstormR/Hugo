About JSON

  • JSON 规范：
    • 数据在键值对中。
    • 数据由逗号分隔。
    • 花括号保存对象。
    • 方括号保存数组。
  • JSON 值：
    • null
    • 数字
    • 布尔值
    • 字符串（双引号 " "）
    • 对象　（花括号 { }）
    • 数组　（方括号 [ ]）
  • JSON 有且仅有 2 种数据结构：对象、数组。
